# orderUpCompiler
Compiler for my OrderUp language

#Make Instructions
###Make Libraries
  From ./compiler directory with g++ installed run:
    ```
    make
    ```
###Make Analyzer Test Program
From ./compiler directory with g++ installed run:
  ```
  make analyzerTest
  ```

###Make Lexer Test Program
From ./compiler directory with g++ installed run:
  ```
  make lexerTest


  ```
#Run Test Program
Run: ```analyzerTest *.oup``` or ```lexerTest *.oup```
